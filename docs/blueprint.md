# **App Name**: Data Canvas

## Core Features:

- Title Page: Display name, role, and brief introduction.
- About Me Section: A page with detailed information about skills and background.
- Work Experience Timeline: Showcase work experience in a chronological timeline. Each entry displays company name, link, and a brief description. Clicking an entry expands to show more details.
- Projects Showcase: Display personal projects with names, technologies used, and descriptions.
- Skills Section: Highlight technical skills and competencies.
- Certifications Display: List certifications achieved.
- Generative AI Assisted Content Structuring: The site uses generative AI to decide the ideal way to order your projects, based on an analysis of the project name, technology, and description, using the tool to incorporate modern recommendations. This will create a more engaging story on your site.

## Style Guidelines:

- Primary color: Dark slate blue (#483D8B) to evoke a sense of sophistication and intelligence.
- Background color: Very light gray (#F0F0F0) for a clean and professional feel.
- Accent color: Soft lavender (#E6E6FA) to provide subtle highlights and interactive elements.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, tech-oriented feel.
- Body font: 'Inter' (sans-serif) for readability and clean presentation.
- Use clean, minimalist icons to represent different skills and technologies.
- Employ a timeline layout for work experience and project history, ensuring readability.
